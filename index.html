Javascript/jQuery flashcards!

Javascript (JS)

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

A symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

Variables

Can be thought of as named containers. You can place data into these containers and then refer to the data simply by naming the container.

Variable naming conventions

Use Camel Case (var = thisIsAVariable). Don't use reserved keywords like "break" or "boolean". Variables should not start with a numeral and they're case sensitive.

Function

A block of code designed to perform a particular task.

Methods

An action that can be performed on object.

Arguments

The real values passed to (and received by) the function.

Parameters

The names listed in the function definition.

Return

A statement that stops the execution of a function and returns a value from that function.

Chaining methods

The act of repeatedly calling one method after another on an object, in one continuous line of code. E.g.: $("#p1").css("color", "red").slideUp(2000).slideDown(2000);

Strings

A datatype that is not a number or a boolean or null.

Booleans

A datatype that is either true or false.

Undefined

A variable that has been declared but has not been assigned a value.

NaN

A variable that has the number type but cannot be read as a number.

Escape

A character (" \ ") that is used to display quotes and other symbols that have other meanings in Javascript.

Alert

A method that displays a window which displays a message.

Comments

Things we write in code that aren't executed in order to make said code easier to read.

jQuery

A library that helps us write less Javascript and do things more easily.

Attributes

What the DOM reads the code as. E.g.: <\body id="page"\> is read as body.id="page".

Prompt

A method that displays a window which asks the user for input.
